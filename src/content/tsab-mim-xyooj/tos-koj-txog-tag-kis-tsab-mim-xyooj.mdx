Yog tias hmo no

Kuv tau ua txhaum rau koj qhov twg

Yuav los thov txim, thov koj txhob rawm chim

Tos kom txog tag kis

<br />

Koj ntxub, koj chim khiav kuv vim dab tsi

Kuv tseem tsis nkag siab

Vim kuv puas yog los koj hloov siab

Wb thiaj tsis sib haum

<br />

[Chorus]

<br />

Tiam sis tag kis yog koj tseem chim

Ua li cas tau

Yog kuv kev txhaum uas tsis paub koj zoo

Yuav ua li cas tag kis

Ntsib los lawm ntau zaug

Lub siab tseem tsis pauv

Tseem niaj zaug uv ntxiv vim hlub koj heev

vim ntseeg koj dhau thiaj mob siab

<br />

Ua tiag siab ntev mog kuv lub siab

Kuv thov uv ntxiv mus

Thiaj tos koj mus txog tag kis

Ntsib koj vim koj siab thov koj kom koj hlub kuv
